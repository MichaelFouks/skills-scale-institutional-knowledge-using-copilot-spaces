# OctoAcme Project Management Docs README

## Overview
OctoAcme's project management processes are designed for transparency, repeatability, and continuous improvement. The docs provide step-by-step guides and checklists for all phases, including initiation, planning, execution, release, risk management, and retrospectives. Core principles include:
- Customer-first, iterative delivery
- Clear ownership for PM, PdM, Developers, QA
- Data-informed decisions and measurable outcomes
- Psychological safety and continuous feedback

See the individual docs below for details on each phase and role.
